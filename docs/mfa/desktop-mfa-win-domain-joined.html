# Overview

In this document we will explore the steps necessary to perform to set up passwordless MFA for your Windows domain-joined computers.

# Prerequisites

We support the following Windows operating systems.

Windows 10YesYes
Windows 11YesYes

TPM requirement

Desktop MFA is supported on Windows machines where TPM is enabled. Please clickÂ here to enable TPM on your PCopen in new window. By default, Windows 11 will have TPM enabled, whereas not all Windows 10 PC will have TPM enabled by default, and if the PC has TPM present and disabled, it can be enabled via BIOS setup.
