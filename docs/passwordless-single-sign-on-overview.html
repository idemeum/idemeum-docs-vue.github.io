# Passwordless Single Sign-On overview

# Overview

You might be familiar with classic Single Sign-Onopen in new window products (Okta, Azure Active Directory, and others) that centralize access to cloud SAMLopen in new window applications - login with username/password once, and then access cloud applications from a consolidated dashboard.

idemeum pushes Single Sign-On to the next level by going beyond web applications.

What is idemeum Passwordless Single Sign-On?

idemeum Single Sign-On is what unifies all idemeum products together to provide one-click access to any company resource. It relies on Passwordless MFA for authentication, so you access everything with biometrics instead of passwords.

Simply put, it does the following:

  1. Unified access - one application catalog to access all you need: SAMLopen in new window/OIDCopen in new window cloud applications, password applications, on-premises applications, SSH servers, RDP desktops, and more.
  2. Management plane - gives you tools to manage users, create groups, apply entitlements, and control access with attribute-based access control (ABAC)open in new window.
  3. Auditing plane - gives you detailed event-based audits and session recording for infrastructure access.
